# MusicPlayer

Android用の音楽プレイヤーアプリです。

## 機能

- **ミニプレイヤーバー**: 画面下部に再生中の曲を表示
- **フルスクリーンプレイヤー**: タップで展開、スワイプダウンで閉じる
- **通知コントロール**: シャッフル、前の曲、再生/一時停止、次の曲、終了
- **プレイリスト対応**: M3Uファイルの読み込み、長押しで削除
- **アーティスト/アルバム表示**: フォルダ形式で階層表示
- **シャッフル/リピート**: 3モード（OFF/全曲/1曲）
- **アルバムアート表示**: 埋め込み画像を表示
- **自動アップデート**: GitHubリリースから更新確認

## セットアップ

1. Android Studioでプロジェクトを開く
2. Build → Make Project
3. Run でデバイスにインストール
